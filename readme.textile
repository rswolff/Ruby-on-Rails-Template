h1. Ruby on Rails Application Template.

h2. Introduction

This is a Ruby on Rails application template.  "Read more":http://guides.rubyonrails.org/2_3_release_notes.html#application-templates about Rails templates. 

h2. Installation

bc. $ rails app_name -m http://github.com/rswolff/Ruby-on-Rails-Template/raw/master/rails_template.rb

Replace _app_name_ with your application name.
